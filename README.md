# Postman Clone

A lightweight web-based API testing tool inspired by Postman. This application allows users to send HTTP requests and inspect API responses directly from the browser through a clean and intuitive interface.

## Live Demo

**Website:** https://postman-clone-toyes3429s-projects.vercel.app

## Features

* Send HTTP requests to APIs
* Support for:

  * GET
  * POST
  * PUT
  * PATCH
  * DELETE
* Custom request headers
* JSON request body support
* View formatted API responses
* Response status monitoring
* User-friendly interface
* Fully browser-based
* Responsive design

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* Axios / Fetch API
* Git & GitHub
* Vercel Deployment


## Getting Started

### Clone the Repository

```bash
git clone https://github.com/toyes3429/Postman--clone.git
```

### Navigate to Project Directory

```bash
cd PostMan-Clone
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Open in Browser

```text
http://localhost:5173
```
